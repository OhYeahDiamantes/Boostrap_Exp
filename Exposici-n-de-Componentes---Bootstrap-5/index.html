Dropdown

Un dropdown (o menú desplegable) es un elemento de interfaz de usuario que permite a los usuarios seleccionar una opción de una lista que se oculta hasta que el usuario hace clic en el botón o activador.

¿Cómo funciona?

  1. Botón Desplegable: Un botón con la clase dropdown-toggle que, al hacer clic, muestra un menú con opciones. Este botón puede tener diferentes estilos (por ejemplo, btn-primary para un botón azul).
  2. Menú Desplegable: Se muestra debajo del botón cuando se activa. Usa la clase dropdown-menu para su estilo y dropdown-item para los elementos del menú.

List Group

Una list group es una lista de elementos que se pueden usar para representar contenido en forma de lista, donde cada elemento puede ser interactivo o no.

¿Cómo funciona?

  1. Elementos de Lista: Cada elemento de la lista es un enlace (<a>) con la clase list-group-item. Los elementos pueden tener diferentes estados:
    • active: Marca el ítem como activo, resaltándolo en la lista.
    • disabled: Indica que el ítem no es interactivo y no se puede seleccionar. También usa tabindex="-1" para evitar que se enfoque con el teclado y aria-disabled="true" para la accesibilidad.

Modal

Un modal es una ventana emergente que se muestra encima del contenido actual, a menudo usada para mostrar información adicional sin redirigir al usuario a una nueva página.

¿Cómo funciona?

  1. Botón de Activación: Un botón que, al hacer clic, abre el modal. Usa el atributo data-bs-toggle="modal" para indicar que debe abrir un modal y data-bs-target para especificar el ID del modal que se debe mostrar.
  2. Contenido del Modal: El modal se compone de tres partes principales:
    • modal-header: Contiene el título del modal y un botón para cerrarlo.
    • modal-body: Contiene el contenido principal del modal.
    • modal-footer: (Opcional) Puede contener botones o acciones adicionales. En este ejemplo no se usa.
  3. Tipos de Tamaño: El modal puede ser pequeño (modal-sm) o grande (modal-lg) dependiendo del contenido y del diseño deseado.
